JavaScript Interpreter
Jsi Home Page
Jsi is a small but feature-full javascript interpreter written in C, and providing:
  • Support for Database, Websocket and JSON
  • An extensive API for both Javascript and C
  • Direct access to C-struct data from Javascript

Industrially, Jsi is used primarily for web-enabled diagnostics, testing and configuration in mass-transit data-acquisition. What distinguishes Jsi from other script languages?

  1. Type-checking for function arguments
  2. A mature design, modelled after Tcl
  3. It's javascript; no need to learn yet-another-scripting-language

Jsi is written in C+/-. It compiles as C, and as native C++ without use of extern C.
